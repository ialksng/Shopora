# Shopora
Designed and developed a production-ready full-stack e-commerce application using the MERN stack, implementing real-world authentication, payments, admin workflows, and AI-driven features.

Key Contributions:
• Built a modern, fully responsive React.js UI <br/>
• Implemented Google Authentication (Firebase) for secure user access <br/>
• Integrated Razorpay for end-to-end payment processing <br/>
• Developed AI-powered product recommendations <br/>
• Created a role-based Admin Dashboard for managing products, orders, users, and inventory <br/>
• Enabled real-time cart, checkout, and stock updates <br/>
• Deployed the application on Render <br/>

Skills: MongoDB · Express.js · React.js · Node.js
