# Shopora
Designed and developed a production-ready full-stack e-commerce application using the MERN stack, implementing real-world authentication, payments, admin workflows, and AI-driven features.

Key Contributions:
• Built a modern, fully responsive React.js UI
• Implemented Google Authentication (Firebase) for secure user access
• Integrated Razorpay for end-to-end payment processing
• Developed AI-powered product recommendations
• Created a role-based Admin Dashboard for managing products, orders, users, and inventory
• Enabled real-time cart, checkout, and stock updates
• Deployed the application on Render

Skills: MongoDB · Express.js · React.js · Node.js
